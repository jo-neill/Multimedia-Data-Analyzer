1. Image Acquisition

A python script was used to retrieve albums of filtered images for training from Flickr. This directly utilized the Flickr API for the acquisition process. Images are the loaded by set for OpenCV processing.

2. Image Formatting

Images acquired are various sizes and aspect ratios. As the primary analysis is undertaken utilizing HSV values, it was determined that formatting all images to be a uniform aspect ratio and size took precedence over preserving aspect ratio. Images are downsized to 250x250 from their current sizes and aspect ratios.

3. HSV Data Extraction

Following formatting, images are then converted from BGR colorspace to HSV. Following much experimentation with histogram feature extraction, we found excellent accuracy could be achieved more quickly by isolating hue and saturation values for the images and computing means.

4. KNN Filter Classifier

After extensively experimenting with SVM models for learning, we established they were not ideal for the dataset and results attempting to be achieved. In the end, supplying a KNN with hue and saturation means proved to be an effective identifier of both the presence and signature of the filters tested.

5. Bulk Analysis with Likes*

In progress. For the full test and subsequent likability analysis, an initial KNN is utilized in a first pass to determine the presence of any detectable filter we are considering. The subsequent layer is the current filter identification model. Successful identifications may then be paired with logged 'like' values from Flickr to determine the most popular or effective filters.
